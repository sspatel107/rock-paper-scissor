# Rock Paper Scissors

A browser-based Rock Paper Scissors game I built for The Odin Project.

## Live demo

[Play it here](https://sspatel107.github.io/rock-paper-scissor/)

## How it works

- I click Rock, Paper, or Scissors to play a round against the computer.
- The score updates each round: `+1 point for you`, `+1 point for computer`, or `no points added` on a tie.
- First to 5 points wins. The buttons then disable and I refresh the page to play again.

## Built with

- HTML, CSS, and vanilla JavaScript.

## Run it

Open `index.html` in a browser.
